# Data Protection Principles

## 1. Be lawful, fair and transparent
- There must be lawful basis for holding and processing data
- The retention and processes of data must be transparently disclosed to affected parties
- Uphold the data subject individual's privacy rights

## 2. Only use personal data for specific purposes
- A privacy impact assessment should be completed each time data is used for a new purpose

## 3. Only collect and keep the minimum of data needed
- Only use the minimal amount of data required to achieve the intended purpose
- A privacy impact assessment should be completed each time additional data is required for a new purpose

## 4. Keep the data secure

## 5. Ensure that the data is accurate
- Data is factual & relevant
- Update and correct data
- Keep data consistent across systems to avoid conflicts

## 6. Only keep the data for as long as necessary

## 7. Be accountable
- Keep audit trail and data keeping records
- Be ready to share data practices and events with data protection regulator
